# TeamPulse
TeamPulse es una aplicación integrada en Microsoft Teams que permite lanzar encuestas internas  de manera rápida y eficiente, utilizando inteligencia artificial para analizar las respuestas en tiempo  real y generar insights accionables

Beneficios Clave
- Integración 100% en Teams: Lanzá encuestas directamente desde los canales y chats.
- Análisis Inteligente: Detección automática de emociones y temas con Azure OpenAI.
- Anonimato garantizado: Mayor honestidad en las respuestas.
- Resultados inmediatos: Recibí insights en tiempo real.
- Exportación de datos: Integración con Power BI para visualización avanzada.

  
Casos de Uso
- Recursos Humanos: Encuestas de clima laboral, detección de burnout, satisfacción.
- Educación: Feedback de clases y evaluación docente.
- Servicios Profesionales: Opiniones post-atención o entrega de proyectos.

Modelo Freemium
Gratis: Hasta 50 respuestas por encuesta, 1 encuesta semanal.
Premium: Encuestas ilimitadas, exportación a Power BI, análisis avanzado con IA
